概念總覽 Concept

設計原則

響應式界面

任何元件都必須符合 Responsive Web Design 的需要，在不同裝置上設計時，除了需在乎設計面積的大小以外，也要將使用該裝置的情境考慮進去。

簡潔易用

UI 元件必須使用最通用、簡易的方式來設計，不用過於複雜的線條與色塊，元件之間的距離必須能夠讓使用者清楚辨別關連性。

高清晰度

所有的元件與圖形盡量使用 HTML/CSS 或 SVG 打造，特殊字型使用 Webfont 技術，讓任何螢幕裝置下，網站都能夠以最高畫質不失真呈現。

規範是活的

此規範本身能讓所有設計師方便瞭解當初設計的概念與想法，並接續取用共同的色彩、元件，以及程式碼，達成網站的一致性。 但針對不同的情境允許自由修正，或是產生新的設計規範收斂回來。
